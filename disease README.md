**Author:** Avika Khanna
**Internship:** CodeAlpha — Artificial Intelligence & Machine Learning

-----

## 📌 Project Overview

This project predicts the possibility of **4 diseases** based on patient data using Machine Learning classification techniques with Indian patient dataset.

-----

## 🔬 Diseases Covered

|Disease                        |Algorithm                          |Dataset Size|
|-------------------------------|-----------------------------------|------------|
|❤️ Heart Disease                |Random Forest + Logistic Regression|1000 records|
|🩸 Diabetes                     |Random Forest + Logistic Regression|1000 records|
|💉 Blood Pressure (Hypertension)|Random Forest + Logistic Regression|1000 records|
|🧠 Brain Tumor                  |Random Forest + Logistic Regression|1000 records|

-----

## ✨ Features

- 🇮🇳 Indian patient dataset with names and cities
- 🤖 Random Forest & Logistic Regression models
- 📊 Accuracy, Precision, Recall, F1-Score, ROC-AUC evaluation
- 📉 Confusion Matrix & Feature Importance charts
- 🔍 Live prediction with confidence score
- 🌐 Beautiful Streamlit web interface

-----

## 🛠️ Technologies Used

- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn
- Streamlit
